WEATHER STATION SETTINGS CREATOR

Hi! This is an app for creating the settings file for your own Weather Station device!

You should go through and set each setting in the Basic Settings portion, and if you are feeling adventurous you can also set the settings in the Advanced Settings portion.

After you set all the necessary settings please click the Create Settings File button. It'll ask you where to save the settings file (it'll be named settings.txt).

Lastly you need to move the new file with the settings to the SD card that you will insert into your device.

If you got here then you're good to enjoy using your Weather Station device!

WiFi SSID

The name of the WiFi network that the device should connect to

WiFi Password

The password to the WiFi network that the device should connect to

TimeZone

The timezone in which the device will be working - used for automatic date and time synchronization

mDNS Hostname

The name that the device should be known as on the local network. If set to e.g. "esp" the device can be accessed as "esp.local" in the browser

Serial Logging Verbosity

The verbosity of the logging the device will do on the serial connection to the PC

WiFi RSSI test period

The period (in milliseconds) of testing the strength of the device's WiFi connection

Data acquisition period

The period (in milliseconds) of getting the data from the device's sensors

Device's temperature sensor select

Which sensor should the device use for temperature measurment

Data Write Interval

Interval in seconds of each data write to the file
